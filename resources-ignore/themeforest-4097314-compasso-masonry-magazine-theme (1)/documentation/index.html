A) Installation - top

In order to begin installing Compasso theme, you need a working copy of WordPress installed on your own server (not on wordpress.com).

Extract the zipped package downloaded from ThemeForest to your desktop. In the extracted package you will find "compasso" folder which is the WordPress theme.

Upload the compasso folder to your wp-content/themes folder. Login to your WordPress admin panel and go to Appearance > Themes. Activate the theme.

Once the theme activated, you can customize Widgets, Menus, Theme Options, and Background

If you want quick setup for your website, you can import XML sample data in the "sample-data" folder. Go to Tools => Import menu and choose WordPress, in upload field choose the xml file and upload it.

B) Pages - top

To create a new page, go to Pages » Add New. The Add New Page screen will be like this:

  1. Enter your page title.
  2. Enter your page content.
  3. Page Attributes:
    • Select a parent page if you want to use this page as a child page.
    • Choose the page template. This theme comes with several different page templates: Categories, Categories Masonry, Contact Form, Grid, List, Masonry, Sitemap.
  4. Below the content editor, you will find custom Meta Boxes. Default page has 2 custom meta boxes: Layout Options and SEO Options.
    • Layout Options:
      • Select a page layout: right sidebar, left sidebar, full width or default. If you choose default, it will use the default layout setting in Theme Options » General
      • Select a sidebar: default sidebar or custom sidebars (generated from Theme Options » Sidebar).
    • SEO Options:
      • Enter a meta description for this page.
      • Enter meta keywords for this page, separated by commas.
      • Check noindex if you don't want this page indexed by search engine.
    When you change the page template, additional custom meta boxes may appear, depends on the type of page. For example, if you choose Sitemap template, it will have Sitemap Options. More of this later (in each page templates).
  5. When you are done, click "Publish" button to save the page.

Default Page - top

Default page is a normal page without special formatting. It will show a standard title and content.

Page Template: List - top

List template will show posts in standard list view (small thumbnail on the left; title, category, rating, published date, comments, excerpt, author and read more link on the right).

In the page admin, list template has the additional List Options meta box. You can set how many posts to show per page and filter posts by category.

Page Template: Grid - top

Grid template will show posts in grid format. Each grid contains 1 post, with a thumbnail (featured image) on the top.

In the page admin, grid template has the additional Grid Options meta box. You can set number of columns (1, 2, 3 or 4), number of posts to show per page, and filter posts by category.

Page Template: Masonry - top

Masonry template will show posts in masonry format. It is like the grid template, but the box will appear in dynamic heights (the featured image will not be cropped).

In the page admin, masonry template has the additional Masonry Options meta box. You can set number of columns (2, 3 or 4), number of posts to show per page, and filter posts by category.

Page Template: Categories - top

Categories template will show posts grouped by category, in 1 column or 2 columns format. Each category in 1 grid can contain more than one post (1 main post with full thumbnail and additional posts with small thumbnail).

In the page admin, categories template has the additional Categories meta box. Click on the Add Category link to add a category. Select a category and set number of posts to show. You can drag the category to reorder.

Page Template: Categories Masonry - top

Categories template will show posts grouped by category, in 3 columns masonry format.

Just like the Categories template, this template also has the Categories meta box to manage which categories to show.

Page Template: Sitemap - top

Sitemap template will show a sitemap with several sections: pages, categories, monthly archive and latest posts.

In the page admin, sitemap template has the additional Sitemap Options meta box. You can enable or disable each section, set the section title, and enter the number of posts (for latest posts section).

  • To enable Pages, check Show Pages, and enter Pages Title.
  • To enable Categories, check Show Categories, and enter Categories Title.
  • To enable Monthly Archives, check Show Monthly Archives, and enter Monthly Archives Title.
  • To enable Latest Posts, check Show Latest Posts, enter Latest Posts Title, and enter the number of Latest Posts.

Page Template: Contact - top

Contact template will show a contact form. This will allow visitors to send a message to your email address. To set email address, subject and success message for the contact form, go to Theme Options » Contact.

Set a page as a Homepage - top

To set a page as a homepage:

  1. Go to Settings » Reading
  2. On Front page displays, choose A static page and select the page from the front page dropdown.
  3. Click Save button.

If you don't set any page as a homepage, your homepage will show latest posts.

C) Posts - top

Default Post - top

To create a new post:

  1. Go to Posts » Add New.
  2. Enter your post title.
  3. Enter your post content.
  4. Choose categories or add a new category.
  5. Add tags, separated by commas.
  6. Set a featured image. Click on "Set featured image" link, a normal upload box will now appear, upload your image and click "Use as featured image".
  7. Fill the SEO Options below the content editor.
  8. When you are done, click "Publish" button to save the post.

Post With Ratings - top

To add ratings on a post, click on the "Enable rating on this post". You can add as many rating as you want, each with a custom criteria. Overall rating will be calculated automatically.

D) Widgets - top

This theme has 4 widget areas for footer and 1 default sidebar. More sidebars can be added from Theme Options » Sidebar.

Here is the location of the widget areas:

You can insert any widgets into each area. To setup widgets, go to Apperance » Widgets and use the drag and drop interface to insert widgets into the desired widget areas.

As an addition to the built-in WordPress widgets, this theme also has 13 custom widgets (Ads 125px, Ads 300px, Comments, Contact, Facebook, Flickr, Posts, Reviews, Social, Subpages, Tabbed Content, Twitter, Video).

Ads 125px - top

Ads 125px widget allows you to show 2 banners, each banner has a 125 pixels width. You can customize the widget title, banner image url and banner link.

Ads 300px - top

Ads 300px widget allows you to show a banner with a 300 pixels width. You can customize the widget title, banner image url and banner link.

Comments - top

Comments widget allows you to show recent comments with an option to show avatar.

Contact - top

Comments widget allows you to show an ajax contact form. To set email address, subject and success message for the contact form, go to Theme Options » Contact.

Facebook - top

Facebook widget allows you to show a facebook likebox.

Flickr - top

Flickr widget allows you to add recent images uploaded to your flickr account. You can customize your Flickr ID, number of photos to show and the display order (latest/random) from the widget settings. To find your Flickr ID, use http://idgettr.com

Posts - top

Posts widget allows you to show posts with these options:

  • Post order: Recent, Popular or Random order.
  • Filter by Category: select All Categories or filter posts by a category.
  • Number of posts to show
  • Show thumbnails: check this option to show small thumbnails.
  • Show ratings (if available): check this option to show a star rating if the post has ratings.
  • Show Dates: check this option to show dates.
  • Show Comment Links: check this option to show comment links.
  • Show Excerpt: show on first item, show on all items, or don't show at all.

Reviews - top

Reviews widgets allows you to show list of latest posts that enable ratings.

Social - top

Social widget allows you to show social links with icons. You can customize icons, link, text and style (horizontal/vertical) from the widget settings. There are 42 icons available that you can choose, with 2 different sizes: 16 and 32 pixels.

Subpages - top

Subpages widget allows you to show child pages of current page or any page. You can customize parent page (Enter a Page ID, or leave empty for current page), sort by and exclude page IDs from the widget settings.

Tabbed Content - top

Tabbed content allows you to show 3 sections: Latest Posts, Popular Posts and Tags. You can customize the number of posts, thumbnails, dates and comment links from the widget settings.

Twitter - top

Twitter widget allows you to add recent tweets from your twitter account. You can customize your Twitter username, number of updates and the follow text from the widget settings.

Because of the latest updates from Twitter API (June 2013), you must now register your app and insert some API keys into the widget settings.

  • Go to https://dev.twitter.com/apps and login with your Twitter account
  • Click "Create a new application" and enter the application details (Name, Description, and Website), enter captcha code and click Create.
  • On the next screen, click "Create my access token".
  • Copy Consumer key, Consumer secret, Access token and Access token secret into the Twitter widget settings.

Video - top

Video widget allows you to show a video and a description. Video can be added from youtube, vimeo or other video sharing sites that provide embed codes. You can customize your video embed code and the description from the widget settings.

E) Menus - top

This theme supports two locations for WordPress menus: Main Menu and Top Menu. Both menu support multi-level dropdown.

To setup the menus, go to Appearance » Menus. Enter a name for your menu and add links to the menu by using the options on the left sidebar. You can add your menu items using the drag and drop functionality. After setting up your menu, select the menu you just created from the theme locations dropdown box and click the Save button.

You can create more custom menus for use within the custom menu widget. Once you have built your menu, save it, and use it from a custom menu widget in Appearance » Widgets

Set home icon for Main Menu

For the Main Menu, you can use a home icon instead of text for the home menu item. Add the "Home" menu item from Appearance » Menus » Custom Links, enter your home URL and click Add to Menu. Set the CSS classes to menu-item-home

If the CSS classes field doesn't appear, click on Screen Options on the top right and check the CSS Classes checkbox.

F) Theme Options - top

To customize this theme using Theme Options, go to Appearance » Theme Options. The Theme Options divided into several sections: General, Header, Sidebar, Slider, Breadcrumbs, Posts, Contact, SEO and Footer.

When you are done making your changes, press the "Save All Settings" button in the bottom-left of the options panel. If you wish to revert to the default settings, press the "Reset" button.

General - top

  • Color Scheme: choose a color from 12 predefined colors, or use the color picker to choose more custom colors.
  • Responsive: this theme uses responsive layout by default. You can disable responsive layout if you don't need it.
  • Default Layout: choose a layout for sidebar alignment. Right sidebar, left sidebar or full width (no sidebar). This will be the default layout, you can change the layout individually on each page settings.
  • Main Logo: check this option to allow an image to be used for the logo. If you disabled this option, your Site Title and Tagline in the WordPress General options will be used.
    In the main logo text field, enter an image url that will be used for the logo. You can also upload an image by clicking on "Upload Image" button. Once the image is uploaded it will give you various options. Click on "Insert into Post" button, the uploaded image url will be inserted into the text field.
  • Custom Login Logo: if you want to change your login page logo (http://www.yourdomain.com/wp-admin), you can enter your image url here, or use the "Upload Image" button.
  • Custom Favicon: enter your favicon image url or use the "Upload Image" button. A favicon is a 16 pixel x 16 pixel image that appears next the URL in most web browsers.
  • Theme Update Notifier: check this option to get notified when a new theme update is available.

Header - top

  • Top Bar: top bar is an area located at the top, that contains menu (Top Menu) on the left and custom text on the right. You can disable top bar by unchecking this option.
  • Header Text: enter a short custom text like introduction to your website, a welcome message, or any text you want. You can also insert shortcodes here, for example if you want to show social media icons.
  • Header Banner: check this option to show banner on the header
  • Banner Image: enter banner image url here, or use "Upload Image" button
  • Banner Link: enter your destination url for your banner

Sidebar - top

This section is for sidebar management. You can add custom sidebars here. Click on the "Add Sidebar" will add a new row with an empty text input. Please enter a unique name for each sidebar. Click on the Delete button will delete the row.

If you want to delete a sidebar, please make sure that the sidebar is empty (no widgets inside it). Sidebars that still have widgets cannot be deleted.

Slider - top

This section is for slider customization. If enabled, the slider will appear on the homepage. Slider items come from posts with a user defined tags (default tag is "featured").

Please note, the post must have a featured image. If the featured image is not found, the post will not show up in the slider. For ideal, please upload an image with dimension of 620 x 350 or higher.

  • Enable Slider: check this to enable the slider
  • Slider Tags: enter tags you want to appear in the slider. If you enter the tag "featured" the slider will look for posts with that tag and show them in the slider. You can enter more than one tag here, separated by commas
  • Animation: choose a transition effect: slide or fade.
  • Slideshow Speed: enter a number to set the speed of the slideshow cycling, in milliseconds
  • Animation Speed: enter a number to set the speed of animations, in milliseconds
  • Pause on Hover: choose true if you want to pause the slideshow when hovering over slider
  • Caption: title and excerpt will be shown in the caption. Additionally, you can show/hide these elements:
    • Category
    • Rating
    • Read More Button

Breadcrumbs - top

  • Enable Breadcrumb Navigation: check this option to show breadcrumbs above the page title.
  • Intro Text: enter text to be inserted before the breadcrumbs.

Posts - top

  • Posts List: this setting is for pages that show many posts like category, tag, search result, author, archive, template list, template grid and template masonry. You can choose to show/hide these elements:
    • Thumbnail
    • Rating
    • Category
    • Date
    • Comments Link
    • Excerpt
    • Author
    • Read More Link
  • Single Post: this setting is for single post view. You can choose to show/hide these elements:
    • Featured Image
    • Tags
    • Author Info
    • Related Posts
    • Share Icons
  • Share Icons: show/hide each icon individually if you enable Share Icons.
  • Enable Ratings: check this option to enable ratings for posts.
  • Enable prettyPhoto: check this option to enable prettyPhoto for thumbnails (when you click on the thumbnails it will open the full image with prettyPhoto). If this option disabled, clicking on the thumbnails will open the post.

Contact - top

This section is for contact page template and contact widget.

  • Email address: enter your email address where you would like to receive message from the contact form, or leave blank to use admin email.
  • Subject: enter a message subject. You can use {name} as a variable for Sender Name.
  • Success message: enter a message to be shown after contact form submitted.

SEO - top

This section is for SEO Settings.

  • Homepage Meta Description: enter your homepage meta description.
  • Homepage Meta Keywords: enter your homepage meta keywords, separated by commas.
  • Noindex: check these options to enable/disable noidex in categories, tags, and archives pages.
  • Enable in post/page: check this option to enable SEO settings in post/page. If you already use a 3rd party SEO plugins, make sure you uncheck this option.

Footer - top

  • Footer Widget Area: check this option to enable footer widget area.
  • Footer Text 1: enter a text like copyright information or any text you want.
  • Footer Text 2: enter a text like copyright information or any text you want.
  • Footer Scripts: enter your custom footer scripts (such as Google Analytics or other tracking code) here. It will be inserted before the closing body tag.

H) Translation - top

To translate this theme, you can use an application called "PoEdit" and you can download it for FREE here: http://www.poedit.net/. Once you installed it:

  • Open the lang directory and copy the default.po into a new file, for example id_ID.po for Indonesian. Languages list can be found here: http://codex.wordpress.org/WordPress_in_Your_Language
  • Edit id_ID.po with Poedit and add translations in this file. After this file saved, id_ID.mo file will be generated. Upload the .mo and .po files into your compasso/lang folder on your live site.
  • Open wp-config.php and edit the WPLANG definition into: 
    define('WPLANG', 'id_ID');

I) Using Child Themes - top

Note: this is for advanced users

If you want to make modifications on this theme, it is recommended to use a child theme. This theme comes with a sample child theme that you can use. In the theme package you can find "compasso-child" folder. Upload this folder to your wp-content/themes folder, and activate it from the Appearance » Themes page.

Once activated, you can start editing inside this "compasso-child" folder. This child theme includes sample on how to override parent theme's style (style.css) and function (functions.php). By using a child theme, you will leave the parent theme unmodified, so when a theme update is available, you can simply replace the parent theme without worry about your modified code. For more info about using child themes, please visit this link: http://codex.wordpress.org/Child_Themes
